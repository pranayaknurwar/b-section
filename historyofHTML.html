HISTORY OF HTML

HyperText Markup Language (HTML) is a markup language used to create web pages.
It was created in 1990 by Sir Tim Berners-Lee, a British computer scientist who is
credited with inventing the World Wide Web. Webpages can include writing, links,
pictures, and even sound and video.

Tags

HTML uses "elements" to let the browser know how a webpage is made of. Elements
are shown as "tags" in the code, written with angle brackets: .
Tags usually come in pairs: an opening tag defines the start of a block of content
and a closing tag defines the end of that block of content. There are many kinds of
tags, and each one has a different purpose
